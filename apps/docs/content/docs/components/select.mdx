A Tailwind CSS select component for choosing from a list of options.

```html preview
<!-- START https://startingpointui.com/examples/docs/select/1 -->
<div class="w-full max-w-xs">
  <select class="select">
    <option value="">Select a country</option>
    <option value="us">United States</option>
    <option value="uk">United Kingdom</option>
    <option value="ca">Canada</option>
    <option value="au">Australia</option>
  </select>
</div>
<!-- END https://startingpointui.com/examples/docs/select/1 -->
```

---

## With label

Pair with a [Label](/docs/components/label) for accessible form fields. Connect them using matching `for` and `id` attributes so clicking the label focuses the select and screen readers announce it correctly.

```html preview
<!-- START https://startingpointui.com/examples/docs/select/2 -->
<div class="w-full max-w-xs">
  <div class="grid gap-3">
    <label class="label" for="country">Country</label>
    <select class="select" id="country" name="country">
      <option value="">Select a country</option>
      <option value="us">United States</option>
      <option value="uk">United Kingdom</option>
      <option value="ca">Canada</option>
      <option value="au">Australia</option>
    </select>
  </div>
</div>
<!-- END https://startingpointui.com/examples/docs/select/2 -->
```

## Disabled

Add `disabled` to prevent interaction. The select appears faded automatically.

```html preview
<!-- START https://startingpointui.com/examples/docs/select/4 -->
<div class="w-full max-w-xs">
  <div class="grid gap-3">
    <label class="label" for="country">Country</label>
    <select class="select" id="country" name="country" disabled="">
      <option value="">Select a country</option>
      <option value="us">United States</option>
    </select>
  </div>
</div>
<!-- END https://startingpointui.com/examples/docs/select/4 -->
```

## Validation

Add `aria-invalid="true"` to show the select in an error state. Connect an error message using `aria-describedby` on the select, pointing to the `id` of the error element. This lets screen readers announce the error when the select is focused.

```html preview
<!-- START https://startingpointui.com/examples/docs/select/5 -->
<div class="w-full max-w-xs">
  <div class="grid gap-3">
    <label class="label" for="country">Country</label>
    <select
      class="select"
      id="country"
      name="country"
      aria-invalid="true"
      aria-describedby="country-error"
    >
      <option value="">Select a country</option>
      <option value="us">United States</option>
    </select>
    <p id="country-error" class="text-xs text-destructive">
      Please select a country
    </p>
  </div>
</div>
<!-- END https://startingpointui.com/examples/docs/select/5 -->
```

## How it works

The select component is a CSS-only utility class applied to a native `<select>` element.

### Structure

Use the `.select` class on a `<select>` element. Pair it with a `.label` element connected via `for` and `id`:

```html
<label class="label" for="country">Country</label>
<select class="select" id="country">
  <option value="">Select a country</option>
  <option value="us">United States</option>
</select>
```

### Disabled state

Add `disabled` to the select:

```html
<select class="select" disabled>
  <option>Disabled</option>
</select>
```

### Validation

Add `aria-invalid="true"` to show the select in an error state:

```html
<select class="select" aria-invalid="true">
  <option value="">Select an option</option>
</select>
```

### Class reference

| Class         | Description         |
| ------------- | ------------------- |
| `select`      | Base select styles  |
| `select-sm`   | Small size variant  |
| `select-base` | Base size (default) |
| `select-lg`   | Large size variant  |
