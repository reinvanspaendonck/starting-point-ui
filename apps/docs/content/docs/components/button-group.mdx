A Tailwind CSS button group component for grouping related [buttons](/docs/components/button) together.

```html preview
<!-- START https://startingpointui.com/examples/docs/button-group/1 -->
<div class="btn-group">
  <button class="btn btn-outline">Left</button>
  <button class="btn btn-outline">Center</button>
  <button class="btn btn-outline">Right</button>
</div>
<!-- END https://startingpointui.com/examples/docs/button-group/1 -->
```

---

## Sizes

Button groups work with all [button sizes](/docs/components/button#sizes).

```html preview
<!-- START https://startingpointui.com/examples/docs/button-group/2 -->
<div class="flex flex-col gap-4">
  <div class="btn-group">
    <button class="btn btn-outline btn-sm">Left</button>
    <button class="btn btn-outline btn-sm">Center</button>
    <button class="btn btn-outline btn-sm">Right</button>
  </div>
  <div class="btn-group">
    <button class="btn btn-outline">Left</button>
    <button class="btn btn-outline">Center</button>
    <button class="btn btn-outline">Right</button>
  </div>
  <div class="btn-group">
    <button class="btn btn-outline btn-lg">Left</button>
    <button class="btn btn-outline btn-lg">Center</button>
    <button class="btn btn-outline btn-lg">Right</button>
  </div>
</div>
<!-- END https://startingpointui.com/examples/docs/button-group/2 -->
```

## Vertical

Stack buttons vertically using `btn-group-vertical`.

```html preview
<!-- START https://startingpointui.com/examples/docs/button-group/3 -->
<div class="btn-group btn-group-vertical">
  <button class="btn btn-outline btn-icon" aria-label="Increase">
    <i class="ri-add-fill"></i>
  </button>
  <button class="btn btn-outline btn-icon" aria-label="Decrease">
    <i class="ri-subtract-fill"></i>
  </button>
</div>
<!-- END https://startingpointui.com/examples/docs/button-group/3 -->
```

## With separator

Add visual separation between buttons using `btn-group-separator`. Recommended for variants without borders like `btn-secondary`.

```html preview
<!-- START https://startingpointui.com/examples/docs/button-group/4 -->
<div class="btn-group">
  <button class="btn btn-secondary">Save</button>
  <div class="btn-group-separator"></div>
  <button class="btn btn-secondary btn-icon" aria-label="More options">
    <i class="ri-arrow-down-s-line"></i>
  </button>
</div>
<!-- END https://startingpointui.com/examples/docs/button-group/4 -->
```

## With text addon

Use `btn-group-text` for non-interactive text labels alongside inputs.

```html preview
<!-- START https://startingpointui.com/examples/docs/button-group/5 -->
<div class="btn-group">
  <span class="btn-group-text">https://</span>
  <input class="input" placeholder="example.com" />
</div>
<!-- END https://startingpointui.com/examples/docs/button-group/5 -->
```

## How it works

The button group component uses `.btn-group` as a container to visually connect related buttons.

### Structure

Wrap buttons in a `.btn-group` container:

```html
<div class="btn-group">
  <button class="btn btn-outline">Left</button>
  <button class="btn btn-outline">Right</button>
</div>
```

### Vertical layout

Add `.btn-group-vertical` for a vertical stack:

```html
<div class="btn-group btn-group-vertical">
  <button class="btn btn-outline">Top</button>
  <button class="btn btn-outline">Bottom</button>
</div>
```

### Class reference

| Class                  | Description                      |
| ---------------------- | -------------------------------- |
| `btn-group`            | Container for grouping buttons   |
| `btn-group-horizontal` | Horizontal layout (default)      |
| `btn-group-vertical`   | Vertical layout                  |
| `btn-group-separator`  | Visual separator between buttons |
| `btn-group-text`       | Non-interactive text addon       |
