A Tailwind CSS textarea component for multi-line text input.

```html preview
<!-- START https://startingpointui.com/examples/docs/textarea/1 -->
<div class="w-full max-w-xs">
  <textarea class="textarea" placeholder="Tell us what you think..."></textarea>
</div>
<!-- END https://startingpointui.com/examples/docs/textarea/1 -->
```

---

## With label

Pair with a [Label](/docs/components/label) for accessible form fields. Connect them using matching `for` and `id` attributes so clicking the label focuses the textarea and screen readers announce it correctly.

```html preview
<!-- START https://startingpointui.com/examples/docs/textarea/2 -->
<div class="w-full max-w-xs">
  <div class="grid gap-3">
    <label class="label" for="message">Message</label>
    <textarea
      class="textarea"
      id="message"
      name="message"
      placeholder="Tell us what you think..."
    ></textarea>
  </div>
</div>
<!-- END https://startingpointui.com/examples/docs/textarea/2 -->
```

```html preview

    ></textarea>
  </div>
</div>
<!-- END https://startingpointui.com/examples/docs/textarea/3 -->
```

## Validation

Add `aria-invalid="true"` to show the textarea in an error state. Connect an error message using `aria-describedby` on the textarea, pointing to the `id` of the error element. This lets screen readers announce the error when the textarea is focused.

```html preview
<!-- START https://startingpointui.com/examples/docs/textarea/4 -->
<div class="w-full max-w-xs">
  <div class="grid gap-3">
    <label class="label" for="bio">Bio</label>
    <textarea
      class="textarea"
      id="bio"
      name="bio"
      placeholder="Write a short bio..."
      aria-invalid="true"
      aria-describedby="bio-error"
    ></textarea>
    <p id="bio-error" class="text-xs text-destructive">
      Bio must be at least 10 characters
    </p>
  </div>
</div>
<!-- END https://startingpointui.com/examples/docs/textarea/4 -->
```

## How it works

The textarea component is a CSS-only utility class applied to a native `<textarea>` element.

### Structure

Use the `.textarea` class on a `<textarea>` element. Pair it with a `.label` element connected via `for` and `id`:

```html
<label class="label" for="message">Message</label>
<textarea class="textarea" id="message" placeholder="Your message"></textarea>
```

### Disabled state

Add `disabled` to the textarea:

```html
<textarea class="textarea" disabled></textarea>
```

### Validation

Add `aria-invalid="true"` to show the textarea in an error state:

```html
<textarea class="textarea" aria-invalid="true"></textarea>
```
