A Tailwind CSS popover component for displaying rich content in a floating panel.

```html preview
<!-- START https://startingpointui.com/examples/docs/popover/1 -->
<div class="popover">
  <button
    class="btn btn-outline"
    type="button"
    data-sp-toggle="popover"
    aria-expanded="false"
  >
    Open Popover
  </button>
  <div class="popover-content">
    <div class="popover-header">
      <h4 class="popover-title">Dimensions</h4>
      <p class="popover-description">Set the dimensions for the layer.</p>
    </div>
    <div class="flex flex-col gap-3">
      <div class="flex items-center gap-4">
        <label class="label w-20 shrink-0">Width</label>
        <input class="input" type="text" value="100%" />
      </div>
      <div class="flex items-center gap-4">
        <label class="label w-20 shrink-0">Height</label>
        <input class="input" type="text" value="25px" />
      </div>
      <div class="flex items-center gap-4">
        <label class="label w-20 shrink-0">Max. Width</label>
        <input class="input" type="text" value="300px" />
      </div>
      <div class="flex items-center gap-4">
        <label class="label w-20 shrink-0">Max. Height</label>

</div>
```

## How it works

The popover component uses a small JavaScript module that handles opening, closing, positioning, and focus trapping.

### Structure

A popover consists of three parts:

1. **`.popover`** - Container that wraps the trigger and content
2. **`[data-sp-toggle="popover"]`** - Button that opens/closes the popover
3. **`.popover-content`** - The floating content panel

```html
<div class="popover">
  <button type="button" data-sp-toggle="popover" aria-expanded="false">
    Open Popover
  </button>
  <div class="popover-content">
    <div class="popover-header">
      <h4 class="popover-title">Title</h4>
      <p class="popover-description">Description text here.</p>
    </div>
    <!-- Any content -->
  </div>
</div>
```

### Opening and closing

Use data attributes to control the popover without writing JavaScript.

Add `data-sp-toggle="popover"` to the trigger button. The JavaScript will find the `.popover-content` sibling and toggle it:

```html
<button data-sp-toggle="popover" aria-expanded="false">Open</button>
```

Clicking outside the popover or pressing Escape closes the content panel. When opened, focus moves to the first focusable element inside the content.

### Focus trapping

When the popover is open, pressing Tab and Shift+Tab cycles focus through the focusable elements inside `.popover-content`. Focus wraps from the last element back to the first, and vice versa. This keeps keyboard users within the popover until they explicitly close it with Escape.

### Animation

The popover includes a default fade and zoom animation. When opened, the JavaScript sets `data-state="open"` on the content element.

To customize animations, add `no-animation` to disable the defaults and use your own classes with `data-[state=open]:` selectors:

```html
<div
  class="popover-content no-animation data-[state=open]:animate-in data-[state=open]:slide-in-from-top-2"
>
  <!-- Custom slide animation -->
</div>
```

### Accessibility

The popover JavaScript module provides focus trapping and proper ARIA attributes.

For proper accessibility, add `aria-expanded="false"` to the trigger button:

```html
<button data-sp-toggle="popover" aria-expanded="false">Open</button>
```

The JavaScript will automatically update `aria-expanded` when the popover opens and closes.

#### Keyboard navigation

| Key               | Action                                                                        |
| ----------------- | ----------------------------------------------------------------------------- |
| `Enter` / `Space` | Opens/closes popover when trigger is focused                                  |
| `Escape`          | Closes the popover and returns focus to the trigger                           |
| `Tab`             | Cycles focus through focusable elements within the popover content            |
| `Shift + Tab`     | Cycles focus backward through focusable elements within the popover content   |

### Class reference

All available classes for the popover component.

| Class                 | Description                                       |
| --------------------- | ------------------------------------------------- |
| `popover`             | Container for the popover                         |
| `popover-content`     | The floating content panel                        |
| `popover-header`      | Optional header wrapper for title and description |
| `popover-title`       | Title text                                        |
| `popover-description` | Description text                                  |
| `no-animation`        | Disables default animation on popover-content     |

```html
<div class="popover">
  <button data-sp-toggle="popover">Open</button>
  <div class="popover-content">
    <div class="popover-header">
      <h4 class="popover-title">Title</h4>
      <p class="popover-description">Description</p>
    </div>
  </div>
</div>
```

### Data attributes

All data attributes for the popover component.

| Attribute                   | Element     | Description                                 |
| --------------------------- | ----------- | ------------------------------------------- |
| `data-sp-toggle="popover"`  | Button      | Toggles the popover                         |
| `data-sp-placement`         | `.popover`  | Content position (default: `bottom`)        |
| `data-state`                | Content     | Set to `open` when content is visible       |

```html
<div class="popover" data-sp-placement="bottom-start">
  <button data-sp-toggle="popover" aria-expanded="false">Open</button>
  <div class="popover-content">
    <div class="popover-header">
      <h4 class="popover-title">Title</h4>
      <p class="popover-description">Description</p>
    </div>
  </div>
</div>
```
