A Tailwind CSS combobox component that combines a trigger button with a searchable, keyboard-navigable list. Supports single and multi-select.

```html preview
<!-- START https://startingpointui.com/examples/docs/combobox/1 -->
<div class="combobox w-56">
  <button
    class="combobox-trigger w-full"
    type="button"
    data-sp-toggle="combobox"
    aria-expanded="false"
  >
    <span class="combobox-value">Select a framework...</span>
    <i class="ri-expand-up-down-line text-muted-foreground"></i>
  </button>
  <div class="combobox-menu w-56" role="listbox">
    <div class="combobox-search">
      <i class="ri-search-line"></i>
      <input
        class="combobox-input"
        type="text"
        placeholder="Search framework..."
      />
    </div>
    <div class="combobox-list">
      <div class="combobox-item" role="option" tabindex="0">
        <input
          type="radio"
          class="sr-only"
          tabindex="-1"
          name="framework"
          value="next"
        />
        Next.js
      </div>
      <div class="combobox-item" role="option" tabindex="0">
        <input
          type="radio"
          class="sr-only"
          tabindex="-1"
          name="framework"
          value="sveltekit"
        />
        SvelteKit
      </div>
      <div class="combobox-item" role="option" tabindex="0">
        <input
          type="radio"
          class="sr-only"
          tabindex="-1"
          name="framework"
          value="nuxt"
        />
        Nuxt.js
      </div>
      <div class="combobox-item" role="option" tabindex="0">
        <input
          type="radio"
          class="sr-only"
          tabindex="-1"
          name="framework"
          value="remix"
        />
        Remix
      </div>
      <div class="combobox-item" role="option" tabindex="0">
        <input
          type="radio"
          class="sr-only"
          tabindex="-1"
          name="framework"
          value="astro"
        />
        Astro
      </div>
    </div>
    <div class="combobox-empty">No framework found.</div>
  </div>
</div>
<!-- END https://startingpointui.com/examples/docs/combobox/1 -->
```

## How it works

The combobox component uses a small JavaScript module that handles opening, closing, filtering, selecting, positioning, and keyboard navigation.

### Structure

A combobox consists of these parts:

1. **`.combobox`** - Container that wraps the trigger and menu
2. **`.combobox-trigger`** - Button that opens and closes the menu
3. **`.combobox-value`** - Span inside the trigger showing the selected label or placeholder
4. **`.combobox-menu`** - The floating panel with search input and item list
5. **`.combobox-list`** - Scrollable list of options
6. **`.combobox-item`** - Individual selectable option containing a radio or checkbox input

Each item contains a visually hidden `<input type="radio">` (single select) or `<input type="checkbox">` (multi-select) with a `name` and `value` for form submission.

```html
<div class="combobox">
  <button
    class="combobox-trigger"
    type="button"
    data-sp-toggle="combobox"
    aria-expanded="false"
  >
    <span class="combobox-value">Select an option...</span>
    <i class="ri-expand-up-down-line"></i>
  </button>
  <div class="combobox-menu" role="listbox">
    <div class="combobox-search">
      <i class="ri-search-line"></i>
      <input class="combobox-input" type="text" placeholder="Search..." />
    </div>
    <div class="combobox-list">
      <div class="combobox-item" role="option" tabindex="0">
        <input
          type="radio"
          name="option"
          value="a"
          class="sr-only"
          tabindex="-1"
        />
        Option A
      </div>
    </div>
    <div class="combobox-empty">No results found.</div>
  </div>
</div>
```

### Opening and closing

Add `data-sp-toggle="combobox"` to the trigger button. The JavaScript finds the `.combobox-menu` inside the same `.combobox` container and toggles it.

Clicking outside the combobox or pressing `Escape` closes the menu. For single select, clicking an item selects it and closes the menu. For multi-select, the menu stays open. The search input is cleared and items are reset when the menu closes.

For programmatic control, use the global `sp.combobox` module:

```js
const combobox = document.querySelector(".combobox");
const item = combobox.querySelector(".combobox-item");

sp.combobox.open(combobox);
sp.combobox.close(combobox);
sp.combobox.toggle(combobox);
sp.combobox.select(combobox, item);
sp.combobox.filter(combobox, "query");
```

### Filtering

Typing in the search input automatically hides items whose text content does not match the query (case-insensitive). The `.combobox-empty` element becomes visible when no items match.

To filter programmatically, call `sp.combobox.filter`:

```js
const combobox = document.querySelector(".combobox");
sp.combobox.filter(combobox, "next");
```

### Selection

When an item is selected, the module checks the item's radio or checkbox input and updates `aria-selected` on the item. The `.combobox-value` text is updated with the selected label.

A native `change` event is dispatched from the input and bubbles up through the DOM. Listen for it on the combobox, a parent form, or any ancestor:

```js
document.querySelector(".combobox").addEventListener("change", (e) => {
  console.log(e.target.name, e.target.value, e.target.checked);
});
```

### Placeholder

The `.combobox-value` span shows muted text when no `data-placeholder` attribute is present (the default state). When a selection is made, the original text is stored in `data-placeholder` and the trigger text updates to the selected label.

For pre-selected values, set `data-placeholder` to the placeholder text and the trigger text to the selected label:

```html
<span class="combobox-value" data-placeholder="Select a framework...">
  SvelteKit
</span>
```

### Animation

The combobox menu includes a default fade and zoom entrance animation. When opened, the JavaScript sets `data-state="open"` on the menu.

To disable the default animation and apply your own, add `no-animation` to `.combobox-menu`:

```html
<div
  class="combobox-menu no-animation data-[state=open]:animate-in data-[state=open]:slide-in-from-top-2"
>
  ...
</div>
```

### Placement

Use `data-sp-placement` on `.combobox` to control the menu position relative to the trigger (default is `bottom-start`). Use `data-sp-offset` to set the distance in pixels (default is `4`):

```html
<div class="combobox" data-sp-placement="bottom-end" data-sp-offset="8">
  ...
</div>
```

Supported placement values follow the [Floating UI](https://floating-ui.com/docs/computePosition#placement) convention: `top`, `top-start`, `top-end`, `bottom`, `bottom-start`, `bottom-end`, `left`, `left-start`, `left-end`, `right`, `right-start`, `right-end`.

### Accessibility

Add `aria-expanded="false"` to the trigger button. The JavaScript updates it automatically when the menu opens and closes. Items should have `role="option"` and `tabindex="0"` so they can receive keyboard focus:

```html
<button
  class="combobox-trigger"
  data-sp-toggle="combobox"
  aria-expanded="false"
>
  ...
</button>
...
<div class="combobox-item" role="option" tabindex="0">
  <input type="radio" name="option" value="a" class="sr-only" tabindex="-1" />
  Option A
</div>
```

#### Keyboard navigation

| Key               | Action                                           |
| ----------------- | ------------------------------------------------ |
| `Enter` / `Space` | Opens the menu when trigger is focused           |
| `Escape`          | Closes the menu and returns focus to the trigger |
| `ArrowDown`       | Move focus to the next visible item              |
| `ArrowUp`         | Move focus to the previous visible item          |
| `Home`            | Move focus to the first visible item             |
| `End`             | Move focus to the last visible item              |
| `Enter` / `Space` | Selects the focused item                         |

### Class reference

| Class                   | Description                                                        |
| ----------------------- | ------------------------------------------------------------------ |
| `combobox`              | Container that wraps the trigger and menu                          |
| `combobox-trigger`      | Button that opens and closes the menu                              |
| `combobox-trigger-sm`   | Small trigger height (h-8)                                         |
| `combobox-trigger-base` | Base trigger height (h-9, default)                                 |
| `combobox-trigger-lg`   | Large trigger height (h-10)                                        |
| `combobox-value`        | Span inside the trigger for the selected label or placeholder text |
| `combobox-menu`         | Floating panel containing the search input and list                |
| `combobox-search`       | Row containing the search icon and input                           |
| `combobox-input`        | Borderless text input for filtering options                        |
| `combobox-list`         | Scrollable container for the list of options                       |
| `combobox-item`         | Individual selectable option                                       |
| `combobox-label`        | Non-interactive group heading                                      |
| `combobox-separator`    | Horizontal divider between groups                                  |
| `combobox-empty`        | Message shown when no items match the search query                 |
| `no-animation`          | Add to `.combobox-menu` to disable the default open animation      |

### Data attributes

| Attribute                   | Element           | Description                                                  |
| --------------------------- | ----------------- | ------------------------------------------------------------ |
| `data-sp-toggle="combobox"` | Trigger button    | Toggles the combobox menu                                    |
| `data-sp-placement`         | `.combobox`       | Menu placement relative to trigger (default: `bottom-start`) |
| `data-sp-offset`            | `.combobox`       | Distance from trigger in pixels (default: `4`)               |
| `data-placeholder`          | `.combobox-value` | Stores the original placeholder text after first selection   |
| `data-state`                | `.combobox-menu`  | Set to `open` or `closed` during animation                   |
| `aria-disabled="true"`      | `.combobox-item`  | Disables the item visually and functionally                  |
| `aria-selected="true"`      | `.combobox-item`  | Marks the item as currently selected                         |
