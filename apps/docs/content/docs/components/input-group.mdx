A Tailwind CSS input group component for decorating [inputs](/docs/components/input) with icons, text, or other addons.

```html preview
<div class="grid gap-6 w-full max-w-xs">
  <div class="input-group">
    <span class="input-group-addon"><i class="ri-search-line"></i></span>
    <input class="input" placeholder="Search..." />
  </div>
  <div class="input-group">
    <span class="input-group-addon"><i class="ri-mail-line"></i></span>
    <input class="input" type="email" placeholder="Enter your email" />
  </div>
  <div class="input-group">
    <span class="input-group-addon"><i class="ri-bank-card-line"></i></span>
    <input class="input" placeholder="Card number" />
    <span class="input-group-addon"><i class="ri-check-fill"></i></span>
  </div>
  <div class="input-group">
    <input class="input" placeholder="Rate this" />
    <span class="input-group-addon">
      <i class="ri-star-line"></i>
      <i class="ri-information-line"></i>
    </span>
  </div>
</div>
```

---

## Text addons

Use text addons for prefixes, suffixes, or units.

```html preview
<div class="grid gap-6 w-full max-w-xs">
  <div class="input-group">
    <span class="input-group-addon">$</span>
    <input class="input" placeholder="0.00" />
    <span class="input-group-addon">USD</span>
  </div>
  <div class="input-group">
    <span class="input-group-addon">https://</span>
    <input class="input" placeholder="example" />
    <span class="input-group-addon">.com</span>
  </div>
  <div class="input-group">
    <input class="input" placeholder="Enter your username" />
    <span class="input-group-addon">@company.com</span>
  </div>
</div>
```

---

## Disabled

Add `disabled` to the input inside the group. The addons dim automatically.

```html preview
<div class="w-full max-w-xs">
  <div class="input-group">
    <span class="input-group-addon"><i class="ri-search-line"></i></span>
    <input class="input" placeholder="Search..." disabled="" />
  </div>
</div>
```

---

## How it works

The input group component wraps an input with addon elements for icons or text.

### Structure

Wrap an `.input` with `.input-group-addon` elements inside a `.input-group` container:

```html
<div class="input-group">
  <span class="input-group-addon"><i class="ri-search-line"></i></span>
  <input class="input" placeholder="Search..." />
</div>
```

Addons can be placed before, after, or on both sides of the input:

```html
<div class="input-group">
  <span class="input-group-addon">$</span>
  <input class="input" placeholder="0.00" />
  <span class="input-group-addon">USD</span>
</div>
```

### Class reference

| Class               | Description                 |
| ------------------- | --------------------------- |
| `input-group`       | Container for grouped input |
| `input-group-addon` | Addon element (icons, text) |
